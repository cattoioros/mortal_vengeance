# Descriere Mortal Vengeance - Action Role-Playing Game (ARPG)

Mortal Vengeance este un **Action Role-Playing Game (ARPG)** cu **perspectivă izometrică (3/4)**, orientat pe **lupte în timp real**, de tip **Hack and Slash**.

## Structură generală
Gameplay-ul se bazează pe **explorarea de hărți (dungeon-uri)**, care pot fi:
- **Generate procedural**, pentru varietate și rejucabilitate  
- **Predefinite**, pentru zonele cheie ale campaniei

Jucătorul controlează direct personajul și se confruntă cu:
- **Grupuri de monștri**  
- **Inamici de tip Elite**  
- Totul sub un sistem de **Fog of War**

## Progresie
Progresia se realizează prin acumularea de **Puncte de Experiență (XP)**, care permit:
- Investiția în **Atribute** precum *Forță*, *Dexteritate* etc.  
- Deblocarea și îmbunătățirea **Abilităților** dintr-un **Skill Tree complex**

## Mecanici centrale
### Loot System
- Sistemul de **Loot Drop** este bazat pe **RNG**.  
- Monștrii, în special **Boss-ii**, pot arunca:
  - **Arme**
  - **Armuri**
- Obiectele sunt clasificate după **raritate**:
  - *Magic*
  - *Rar*
  - *Legendar*
- Fiecare item are **affix-uri generate aleatoriu**, esențiale pentru **build-ul personajului**.

## Obiectiv
Obiectivul final al fiecărei zone este **confruntarea cu un Boss** cu **mecanici de luptă dedicate**, a cărui înfrângere este necesară pentru **avansarea în campanie**.
